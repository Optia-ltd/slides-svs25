What makes an optimization algorithm?

$ \displaystyle \minimize_{x \in X} f(x) $
Three main ingredients
Ingredient Description
Decision variables $x$ The things we can control
e.g., control signals, mixture components, model parameters, ...
Constraints $X$ A candidate solution $x$ is only valid if $x \in X$
e.g., actuator limits, safety constraint, physical laws, ...
Cost/Objective function $f$ Takes a solution $x$ and returns a number — lower is better
e.g., Energy consumption, system performance, monetary cost, ...
Two key challenges
How to define $x, f, X$?
  • Do we have formulas, simulators, data?
  • May involve machine learning another optimization problem!
How to compute the solution numerically?
  • How much time available? milliseconds → days
  • How much precision required?
